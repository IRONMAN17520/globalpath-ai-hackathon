# Requirements - GlobalPath AI

## Problem Statement
Immigrants and international students struggle with region-specific bureaucracy (visa rules, healthcare enrollment, housing rights, tax obligations), leading to delays, fines, and barriers to education/opportunities.

## User Needs
- Upload official documents (PDF/photo) for parsing
- Receive personalized, step-by-step action plans in plain language
- Get links to trusted resources (Mastersportal.com, EducationUSA.gov, official embassies, DAAD.de)
- Deadline reminders and risk alerts
- Multilingual support (English + Hindi + others)

## Functional Requirements
1. Document upload & extraction (visa notices, acceptance letters, etc.)
2. AI-based plan generation using LLM + policy knowledge
3. Country-specific rules (start: US, Canada, Germany, UK, Australia, India incoming)
4. Resource recommendation engine
5. Usable UI (web/mobile)

## Non-Functional
- Accessible (plain language, multilingual)
- Privacy-focused (no unnecessary data storage)
- Scalable via AWS

## Success Metrics
- Reduce navigation time by 70%
- High user satisfaction for underserved communities
